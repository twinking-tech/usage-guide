Usage Guide

Welcome to the Weather Website! This guide will help you navigate the website and understand each feature to get the most out of your experience.

Getting Started

On the home page, you can view current weather information for any location you choose. This data is fetched from an external weather API.

Using the Search Feature

  • Enter Location: Use the search bar to type in the name of a city or region you want to check the weather for.
  • Submit Search: Press the Search button to retrieve the weather data.
  • Error Handling: If the location is not found, an error message will prompt you to try again.

Current Weather Display

Once you search for a location, you will see the following details:

  • Temperature: The current temperature at the location.
  • Weather Condition: Displays conditions like sunny, cloudy, rainy, etc.
  • Location: Shows the name of the city and country you searched for.

5-Day Forecast

Scroll down on the home page to view the 5-day forecast for your selected location. Each card includes:

  • Daily Highs and Lows: Minimum and maximum temperatures for each day.
  • Weather Icons: Visual indicators of the forecast (e.g., sun, clouds, rain).
  • Scrollable Forecast: On smaller screens, the forecast section is horizontally scrollable.

Responsive Design

This website adjusts its layout based on the device screen size:

  • For tablets, elements resize and reposition for an optimal view.
  • For mobile devices, the site layout simplifies, and you can scroll horizontally to view forecast cards.

We hope this guide helps you enjoy a seamless experience using our weather website!
